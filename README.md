## Features

### 지도 (메인 탭)
- 지도 선택 위치에 핀 고정 기능 → 이미지 업로드, 좋아하는 이미지 하트 표시
- 위치 정보 표시
- 지도 연락처의 주소 → 지도 핀으로 표시
